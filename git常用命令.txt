git reset commit-id --hard
重置到某一个版本，未来的版本也可以
这个命令会清除工作区的修改，所以，需要很小心。

注意：先push到服务器上，然后再reset以前的版本，而不是reset到以前版本，再reset刚刚的版本来还原。
